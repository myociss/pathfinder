## Getting Started

### Prerequisites

Python 3, git, pip3.

### Dependencies

Install Eigen:

```
sudo install libeigen3-dev
```

### Installation

```
git clone https://github.com/myociss/pathfinder
pip3 install ./pathfinder
```

To run tests:

```
cd pathfinder
python3 setup.py test
```
